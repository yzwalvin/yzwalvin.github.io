Developing The Pharaoh's Legacy

The Pharaoh's Legacy is a collaborative game project developed by a team of 8 students. It was built using a custom 2D game engine written in C++, featuring an integrated ImGui interface, scripting support, and a component-based architecture.

As the Product Manager, I worked closely with the team, organizing team meetings and tasks. I also implemented an internal debugging system for identifying runtime issues easily, engineered a particle and animation component system that allows for seamless transitions and visual effects, and also integrated an asset browser to allow drag and drop of files into the engine.
