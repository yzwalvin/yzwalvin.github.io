Final Year Project (Polytechnic)
Oct 2020 - Feb 2021

This project's aim is to design and develop a web application that allows educators to easily generate, assign, and manage Python exercises.

Students can access these papers and attempt them online, and access will be restricted after the time allocated has been used up.

Developing

Over 20 weeks, our team of 3 members built a working web application that utilises C# HTML in ASP.NET while leveraging MySQL as a database to store all information.

As the team leader and programmer, I designed and implemented the core architecture and functionality of the python quiz generator while organising team meetings, and collaborating with team members to ensure seamless feature integration and maintain overall system cohesion.
